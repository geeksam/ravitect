To install this...

From the command line:

  $ cd ~/Library/Application\ Support/TextMate/Bundles/
  $ git clone https://github.com/geeksam/ravitect Ravitect.tmbundle

If TextMate is open, switch back to it and choose "Bundles -> Bundle Editor -> Reload Bundles"

You may want to add some syntax-highlighting entries with the following scope selectors (TextMate -> Preferences, Fonts & Colors):

markup.other.ravitect.annotation.brackets
markup.other.ravitect.annotation.parens
variable.other.ravitect.vip_word
variable.other.ravitect.action_or_clarifier
